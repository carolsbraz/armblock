O que é a IDE e para que serve?

A IDE do Arduino é o ambiente de desenvolvimento do software, possibilitando a programação e os comandos que o microcontrolador deve executar.

Ao abrir o IDE é mostrada uma estrutura em linguagem de programação C com 2 funções, setup() e loop(). É na função setup() que são programadas as inicializações e configurações do microcontrolador Arduíno, como a definição da pinagem dos componentes. A função loop() é responsável por executar o código de controle, por exemplo, do funcionamento de um braço robótico
