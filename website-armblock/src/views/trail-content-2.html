O que é um robô?


Você deve estar se perguntando “Bom, eu já sei o quê é robótica, mas afinal o quê é um robô?”.

Um robô é um sistema integrado composto por sensores, manipuladores, sistemas de controle, fonte de energia e um software (programa de computador), os quais trabalham em conjunto a fim de realizar uma tarefa.
