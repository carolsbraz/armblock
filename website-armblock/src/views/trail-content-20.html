O que é a IDE e para que serve?



A IDE do Arduino é o ambiente de desenvolvimento do software, possibilitando a programação e os comandos que o microcontrolador deve executar.

IDE do Arduino

Fonte: Autoria própria (2021)

A primeira estrutura logo que você abre a IDE do Arduino é a função setup (). Na função setup () é onde são feitas as inicializações e configurações do microcontrolador Arduino, sendo descrito onde estão ligados os componentes Na segunda função denominada loop () é onde será escrito o código em si. Onde será feito um loop infinito que fará rodar o código enquanto o Arduino estiver ligado.

Opções da barra de ações da IDE Arduino

Fonte: Autoria própria (2021)

  • Compilar– Está função serve para verificar se há algum erro. Se houver algum erro irá ser mostrado (zona inferior da interface);
  • Enviar- Este botão serve para compilar e enviar o código para o Arduino;
  • Novo ficheiroEste botão serve para criar um ficheiro;
  • Abri– Ao clicar neste botão irá aparecer uma janela para navegarmos até ao ficheiro que pretendemos abrir;
  • Guardar- Como o nome diz, este botão serve para guardarmos o nosso ficheiro. Aqui temos o nome do ficheiro;
  • Monitor serial- Ao clicarmos aqui, irá aparecer uma janela onde são mostrados alguns dados.
