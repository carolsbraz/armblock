O que são sensores?



Você deve estar se perguntando o que é um sensor? Para quê serve?”.

Bom, um sensor nada mais é que um dispositivo que mede o valor de uma grandeza física, como temperatura, distância, velocidade e pressão. Os robôs que trabalham inseridos em ambientes reais, estáticos (fixos) ou dinâmicos, estão dotados de sensores que lhes permitem adquirir informações de como agir no ambiente.

Clicando no bot, você será direcionado para um pdf sobre sensores e clicando no sensor, você será direcionado para uma página.
